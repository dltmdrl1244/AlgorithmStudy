# 문제

링크 : [1633 최고의 팀 만들기](https://www.acmicpc.net/problem/1633)

해결여부 : yes

# 문제 풀이

- 한 선수는 흑팀에 속하거나, 백팀에 속하거나, 아무 팀에도 속하지 않는다
- 이 때 각 팀에 속하려면, 그 팀에서 뽑을 수 있는 기회가 남아 있어야 한다
- 어느 팀에 속했다면 그 팀의 횟수를 1 차감하고, 아니라면 차감하지 않고 idx를 다음으로 넘겨 다음 선수를 확인한다.

# 마무리

브루트포스로 갈겼는데 됐다.
사람수가 최대 1000명이라서 안 될 줄 알았는데, 메모이제이션도 하고 각팀당 최대인원수가 15명이다 보니 가짓수가 그렇게 많이 뻗어 나가지는 않는 모양이다.
