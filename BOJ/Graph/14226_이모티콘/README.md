# 문제
링크 : [14226 이모티콘](https://www.acmicpc.net/problem/14226)

해결여부 : YES

# 문제 풀이
> 응용 BFS 문제
현재 이모티콘 개수, 클립보드에 저장된 개수를 쌍으로 표현하며 2차원 공간을 탐색하는 식으로 생각
최대 길이가 1000이므로 1000x1000 크기의 보드를 만듬 (이정도까지 크게 필요하진 않을 것 같긴 함)

다음 동작은 현재 개수에 클립보드 개수를 추가하는 것, 클립보드 상황을 현재 이모티콘 개수로 덮어쓰는 것, 현재 개수에 하나를 지우는 것 3가지가 있음. 방문여부 확인을 한 후 BFS를 진행하면 됨
