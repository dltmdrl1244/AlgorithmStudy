# 문제
링크 : [14719 빗물](https://www.acmicpc.net/problem/14719)

해결여부 : no

# 문제 풀이
- 특정 칸에 빗물이 고이려면 좌우로 그 칸보다 높은 벽이 있어야 한다
- 그리고 만약 있다면, 빗물이 고이는 양은 (좌우 가장 높은 벽 중 작은 값) - 특정 칸 높이
- 이 과정을 모든 칸에 대해 수행한다. (한 번의 시행당 모든 칸을 보므로 n^2)
