# 문제
링크 : [1451 직사각형으로 나누기](https://www.acmicpc.net/problem/1451)

해결여부 : yes

# 문제 풀이

세 개의 직사각형으로 나누는 방법은 네 가지
1. 가로로 두번 자르기
- 조건 : 세로길이가 3 이상이어야
- 방법 : 자를 인덱스를 조합을 통해 선정. 선택한 인덱스의 위쪽을 자른다고 가정 (선택한 인덱스가 아래쪽 사각형에 속하게)
1~n-1 중에서 2개를 선택

2. 세로로 두번 자르기
- 조건 : 가로길이가 3 이상이어야
- 방법 : 자를 인덱스를 조합을 통해 선정. 선택한 인덱스의 왼쪽을 자른다고 가정 (선택한 인덱스가 오른쪽 사각형에 속하게)
1~m-1 중에 2개 고르기

3. 가로 한번, 세로 한번 자르기
    1) 두줄 차지하는 사각형이 상하로 올 때
    - 조건 : 가로길이가 2 이상이어야
    - 방법 : 먼저 가로로 자를 인덱스를 정한다 (0~n-1 사이)
    그리고 세로 인덱스를 구한다 (0~m-1 사이)
    한번은 위로 한번은 아래로 두가지 케이스

    2) 두줄 차지하는 사각형이 좌우로 올 때
    - 조건 : 세로길이가 2 이상이어야
    - 방법 : 먼저 세로로 자를 인덱스를 정한다 (0~m-1 사이)
    그리고 가로 인덱스를 구한다 (0~n-1 사이)
    한번은 왼쪽, 한번은 오른쪽
