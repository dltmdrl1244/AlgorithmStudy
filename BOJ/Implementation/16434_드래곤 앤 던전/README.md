# 문제
링크 : [16434 드래곤 앤 던전](https://www.acmicpc.net/problem/16434)

해결여부 : yes

# 문제 풀이
먼저 최대 체력을 정하지 말고, 보스방까지 클리어하는 데에 '감소된 HP량' 을 구함
몬스터들과 싸우는 과정은 항상 내가 먼저 선빵을 때리므로 몬스터를 때리는 횟수를 구하고 거기에 -1만큼을 내가 맞는다 (한방에 죽이면 나는 안 맞을 수 있다)
그리고 회복이 최대 체력 이상으로 되지는 않으므로, 변화량이 +가 되지 않게 포션을 먹을 때 min(0) 을 써서 보정

주의해야 할 점은 단순히 클리어 시점의 HP 변화량에 1을 더하는 것이 아니라는 것. 중간에 더 피해를 많이 입었을 수도 있고 회복을 한 것일수도 있기 때문. 예를 들어 몬스터들을 처치하느라 HP가 100이 닳았고 거기에 포션으로 70만큼 회복한 후 보스를 처치했을 때 90이 닳아있었다고 해서 최대 HP가 91가 되면 애초에 끝까지 진행할 수가 없기 때문
즉 각각의 몬스터와의 대전 이후 ans를 중간중간에 계속 갱신해 줘야 한다
