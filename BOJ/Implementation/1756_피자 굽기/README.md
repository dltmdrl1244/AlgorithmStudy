# 문제
링크 : [1756 피자 굽기](https://www.acmicpc.net/problem/1756)

해결여부 : yes

# 문제 풀이
위쪽에서부터 피자를 넣으므로, 아래쪽에 넓은 공간이 있어도 위쪽에 입구가 좁으면 피자를 넣을 수 없다
따라서 위에서부터 반지름을 탐색해 가며 '이 칸에 넣을 수 있는 피자의 최대 크기' 를 기록한다. (현재까지 발견한 최솟값을 기록하면 됨)

그 이후 피자는 순서대로, 오븐은 아래에서부터 탐색하며 이 칸에 피자를 넣을 수 있으면 피자를 하나씩 넣어 간다.
오븐 끝까지 탐색했을 때, 아직 넣을 피자가 남아 있다면 0을 출력, 만약 중간에 피자를 모두 넣었다면 그 지점의 위치를 출력하면 된다.
